# Contributing to Java Interview Programming Problems

Thank you for your interest in contributing to this repository! 🎉
This project aims to help beginners practice Java programming problems commonly asked in technical interviews.

## How to Contribute

Please follow these steps to contribute to this repository:

### 1. Fork the Repository

Click the **Fork** button on the top right corner of the repository to create your own copy.

### 2. Clone the Forked Repository

Clone the repository to your local system.

```
git clone https://github.com/2003MADHAV/Java-Interview-Programming-Problems.git
```

### 3. Create or Add a New Program

Add a new Java program related to:

* Array Problems
* String Problems
* Number Series Problems
* Basic Java Interview Problems
* Pattern

Make sure your program is placed in the correct folder.

Example structure:

```
src/
 ├── arrayProblems/
 ├── stringProblems/
 └── numberSeriesProblem/
```

### 4. Write Clean Code

Please ensure:

* Proper indentation
* Meaningful class names
* Comments explaining the logic

### 5. Commit Your Changes

```
git add .
git commit -m "Added new Java interview problem"
```

### 6. Push the Changes

```
git push origin main
```

### 7. Create a Pull Request

Go to your forked repository on GitHub and click **Compare & Pull Request**.
Submit the pull request so it can be reviewed and merged.

## Contribution Guidelines

* Do not add duplicate problems.
* Keep the code simple and readable.
* Add comments for better understanding.

Thank you for contributing and helping others learn Java! 🚀
